#  关于Matrix-LaTeX-Generator

## 制作原因
因为LaTeX中矩阵的表示方式不够直观，填写繁琐,所以于2018年写了这个相对直观的小App来辅助生成LaTeX代码。于2019年5月重构部分代码后开源。

## 帮助文档?
现在还没有,等功能完善后会添加.
## 开源?
本程序中代码全部使用AGPLv3开源，图形资源（如Logo、按钮UI）**不开源**，作者保留所有权利。
